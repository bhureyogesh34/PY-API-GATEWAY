1. Install requirements

    pip install -r requirements.txt

    pip install fastapi uvicorn sqlalchemy passlib python-jose[cryptography] httpx

2. Run the application

    uvicorn app.main:app --reload

3. Register a user

    curl -X POST "http://localhost:8000/register" -H "Content-Type: application/json" -d '{"username":"testuser","email":"test@example.com","password":"secret"}'

4. Get an access token:

    curl -X POST "http://localhost:8000/token" -H "Content-Type: application/x-www-form-urlencoded" -d "username=testuser&password=secret"

5. Use the API Gateway with the token:

    curl -X GET "http://localhost:8000/product_service/products" -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
    curl -X GET "http://localhost:8000/product_service/products" -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
